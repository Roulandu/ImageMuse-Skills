# Image2 成人魅力与时尚提示词技能包

这个仓库包含 4 个 Codex skill，用于把成人、原创、非露骨的女性魅力写真、时尚人像、内衣/泳装 Lookbook 和失败修复需求，整理成可直接用于 Image2 / GPT Image 2 的中文提示词。

它的目标不是生成软色情或绕过审查，而是把成熟吸引力、服装版型、整体体态、商业审美、镜头语言和高级时尚氛围，转成更稳定、更清晰的图像提示词。

默认主体是“原创成年东方女性，二十多岁及以上”。如果用户明确指定其他安全主体，技能会尊重用户设定。提示词会优先保持成熟、原创、非露骨、非真实人物相似的方向。

如果用户没有指定姿态或镜头，默认会使用自然站姿或坐姿、面向镜头或自然三分之四角度、正视或轻微俯视的 editorial 镜头。回眸、背身、明显侧身、仰拍等方向只在用户明确要求且安全时使用；低机位身体凝视、偷拍感、窥视感、身体局部主导构图会被改写或移除。

## 通用使用规则

- 想把粗糙、低俗、容易失败的想法改成高级版本，用 `$image2-aesthetic-rewriter`。
- 想直接生成成人魅力写真、高级女性人像、时尚 editorial 人像，用 `$image2-adult-glamour-studio`。
- 涉及内衣、泳装、试衣间确认照、Lookbook、fit check，用 `$image2-fashion-lingerie-lookbook`。
- 提示词被拒、生成结果太保守、脸太幼、太擦边、身材或手部变形、不够高级，用 `$image2-risk-debugger`。

使用时可以直接写：`使用 $技能名，帮我……`，然后给出你的主题、风格、服装、场景、画幅比例或遇到的问题。

## 技能详细使用方法

### `$image2-aesthetic-rewriter`

适合把“性感、火辣、撩人、暧昧、容易擦边”的描述，改写成成熟、高级、非露骨、适合 Image2 的中文提示词。它会把欲望化表达转换为整体审美表达，例如成熟自信、优雅姿态、服装剪裁、自然体态、商业人像、editorial 光线和镜头语言。

推荐这样问：

```text
使用 $image2-aesthetic-rewriter，把“性感美女，身材火辣，氛围暧昧”改成高级、克制、能用于 Image2 的中文提示词。
```

常见用途：

- 把粗糙或低俗词汇改成成熟审美语言。
- 把身体局部强调改成整体体态、服装剪裁和自然姿态。
- 把私密、暧昧或容易失败的场景改成时尚、人像、商业或角色设计语境。
- 为后续人像、Lookbook 或修复流程打一个更稳定的基础。

使用注意事项：

- 支持保留成熟吸引力，但不会保留软色情、露骨、未成年感、真实人物性化或绕过安全系统的意图。
- 如果原始方向太高风险，它会保留可支持的部分，并改写成成熟、原创、非露骨的时尚或 editorial 方向。

### `$image2-adult-glamour-studio`

适合直接生成成人原创魅力写真、高级女性人像、成熟美、时尚杂志感、商业 portrait 或 cinematic editorial 风格提示词。重点是脸部气质、服装质感、姿态、灯光、整体轮廓和画面高级感。

推荐这样问：

```text
使用 $image2-adult-glamour-studio，生成一张原创成年东方女性高级人像的中文提示词，清冷、成熟、低饱和、窗边自然光，9:16。
```

常见用途：

- 成人魅力写真
- 高级女性人像
- 成熟女性美
- 时尚杂志感
- 商业人像和 editorial 视觉
- 电影感生活方式人像

使用注意事项：

- 默认主体为原创成年东方女性，二十多岁及以上，不接近真实人物、明星、网红或私人照片。
- 默认使用自然站姿或坐姿、优雅三分之四转身、正视或轻微俯视 editorial 镜头。
- 想要更强张力时，建议描述眼神、姿态节奏、服装结构、光线、色彩和构图，而不是要求“更性感”。

### `$image2-fashion-lingerie-lookbook`

适合生成成人时尚内衣、bralette、泳装、试衣间服装确认照、Lookbook、fit check 和商品展示方向的提示词。它会优先表现服装，而不是身体局部。

推荐这样问：

```text
使用 $image2-fashion-lingerie-lookbook，生成一张原创成年东方女性模特的高级泳装 Lookbook 中文提示词，度假酒店泳池边，重点表现泳装版型、面料和清爽夏日氛围。
```

常见用途：

- 内衣 Lookbook
- 泳装品牌图
- bralette / lingerie / swimwear 商品展示
- 试衣间服装确认照
- 成人时尚 fit check
- 度假或精品店场景的商业目录图

使用注意事项：

- 请把重点放在服装类型、覆盖度、支撑结构、面料、颜色、纹理、版型、搭配和使用场景上。
- 画面应包含脸部、服装、姿态和环境，避免只裁切胸、腰、臀、腿根、内衣细节或私密区域。
- 内衣和泳装必须是非露骨、得体覆盖、商品展示或时尚语境；偷拍、泄露、男友视角、隐藏镜头、私密发送感会被改写。

### `$image2-risk-debugger`

适合修复已经失败或不满意的提示词。你可以给它原提示词、拒绝原因、生成结果描述，或者直接说明问题，例如“被拒了”“太保守”“脸太幼”“太擦边”“身材变形”“手指畸形”“不够高级”“不像真实摄影”。

推荐这样问：

```text
使用 $image2-risk-debugger，下面这条提示词被拒了，请诊断原因并输出一个合规的中文修复版：
……
```

也可以这样问：

```text
使用 $image2-risk-debugger，这条提示词生成结果太保守，不够时尚张力，脸也偏幼，请修复成更成熟、高级、真实摄影感的版本：
……
```

常见用途：

- 被拒绝生成
- 结果太保守
- 脸太幼或有未成年感
- 太擦边或软色情感太强
- 身材、手部、肢体结构异常
- 画面不够高级
- 不像真实摄影或商业人像

使用注意事项：

- 它会优先做最小修复，尽量保留可支持的用户意图。
- 如果原方向太高风险，会改成更安全的时尚、人像、Lookbook 或 editorial 版本。
- 缺失或不安全的镜头、姿态和构图，会被修复为自然姿态、较宽画面、正视或轻微俯视的商业/editorial 镜头。

## 输出格式

4 个技能默认使用简体中文输出。常见字段包括：

```text
意图摘要
已锁定参数
风险/质量说明
最终中文提示词
负面限制词
可选变化参数
```

`$image2-risk-debugger` 还会包含：

```text
修复策略
```

字段含义：

- `意图摘要`：概括安全后的视觉目标。
- `已锁定参数`：列出保留的主体、风格、场景、服装、姿态、镜头、画幅等信息。
- `风险/质量说明`：说明哪些风险被降级，以及画面质量如何提升。
- `最终中文提示词`：可以直接复制到 Image2 / GPT Image 2 使用的主体提示词。
- `负面限制词`：用于约束不希望出现的内容，例如未成年感、真实人物相似、露骨裸露、身体局部特写、低机位身体凝视、手指畸形等。
- `可选变化参数`：给出可以继续调整的方向，例如色调、镜头距离、服装材质、场景、姿态、画幅比例或真实感等级。

## 重要边界

这些技能会保留成人魅力、时尚张力、成熟体态和高级审美，但不会帮助生成或保留以下方向：

- 软色情或露骨色情目标
- 未成年、幼态、学生感、少女感与性感组合
- 真实人物、明星、网红、上传照片或私人形象的性化
- 偷拍、泄露、男友视角、隐藏镜头、窥视、胁迫、醉酒、压迫感亲密互动
- 胸、臀、腰胯、腿根、唇部、私密区域等身体局部主导构图
- 低机位身体凝视、透明裸露、滑落服装、私密暴露
- 绕过审查或规避安全系统的写法

如果原始请求包含这些内容，技能会拒绝或改写相关部分，并把可支持的方向转成成人、原创、非露骨的时尚、人像、Lookbook、商业摄影或角色设计提示词。
