# Image2 女性人像提示词 Skills

生成高级女性人像提示词的 Codex skill。

这个仓库包含 7 个 Image2 / GPT Image 2 中文提示词 skill，用于整理成人、原创、非露骨的女性人像、时尚写真、婚纱摄影写真、内衣/泳装 Lookbook、人物海报/杂志封面、人物专业设定图和失败修复需求。

重点是把面容气质、体态比例、服装版型、场景氛围和镜头语言，整理成更稳定、更清晰、更有商业审美的图像提示词。

默认主体为“原创成年东方女性，20 岁及以上”。如果用户指定其他安全主体，skill 会尊重用户设定。

## 怎么用

直接写：

```text
使用 $技能名，帮我……
```

然后补充主题、风格、服装、场景、画幅比例，或说明遇到的问题。

## 7 个 Skill

### `$image2-aesthetic-rewriter`

把粗糙、低俗、容易失败的想法，改写成成熟、高级、非露骨、适合 Image2 的中文提示词。

```text
使用 $image2-aesthetic-rewriter，把“性感美女，身材火辣，氛围暧昧”改成高级、克制、能用于 Image2 的中文提示词。
```

### `$image2-adult-glamour-studio`

直接生成成人原创女性人像、魅力写真、时尚 editorial、商业 portrait 或电影感人像提示词。

```text
使用 $image2-adult-glamour-studio，生成一张原创成年东方女性高级人像的中文提示词，清冷、成熟、低饱和、窗边自然光，9:16。
```

### `$image2-fashion-lingerie-lookbook`

生成成人时尚内衣、bralette、泳装、试衣间确认照、Lookbook、fit check 和商品展示提示词。重点是服装版型、面料、覆盖度、搭配和商业展示语境。

```text
使用 $image2-fashion-lingerie-lookbook，生成一张原创成年东方女性模特的高级泳装 Lookbook 中文提示词，度假酒店泳池边，重点表现泳装版型、面料和清爽夏日氛围。
```

### `$image2-wedding-portrait`

生成婚纱摄影写真、婚纱照、bridal portrait 和 wedding editorial 中文提示词。默认主体为“20岁出头的中国美人。”，默认表情为“自然的笑容。”，默认 3:4、照片分辨率 8k；婚纱衣着可随机生成，也可由用户指定。

```text
使用 $image2-wedding-portrait，生成默认婚纱摄影写真中文提示词，图片风格选择专业单反相机写真拍摄质感。
```

也可以指定主体、表情、婚纱衣着、身材选项和图片风格：

```text
使用 $image2-wedding-portrait，主体是温婉成熟的中国新娘，表情是含蓄微笑，婚纱为缎面一字肩鱼尾婚纱，身材选项偏大，图片风格选择 iPhone 手机直出拍摄质感。
```

### `$image2-character-poster-cover`

生成完整人物海报、杂志封面、editorial cover、宣传主视觉提示词。成图目标是带原创标题、虚构刊名、副标题、封面线、日期、期号、价格样式和宣传语的成品海报/封面，不默认留空白标题区。

```text
使用 $image2-character-poster-cover，生成一张原创成年东方女性的高级时尚杂志封面中文提示词，冷白摄影棚、银灰色调、成熟自信，标题和封面线随机生成，3:4。
```

### `$image2-character-reference-sheet`

生成人物专业设定图、角色三视图、全身转面图和服装细节设定板提示词。默认在一张横向大图中包含全身正面、侧面、背面视图，以及面部、发型、服装材质、配件、鞋履等局部细节框。

```text
使用 $image2-character-reference-sheet，生成一张原创成年东方女性角色的专业设定图中文提示词，包含全身正面、侧面、背面和服装细节框，浅灰设计板背景，16:9。
```

### `$image2-risk-debugger`

诊断并修复被拒、太保守、脸太幼、太擦边、身材或手部变形、不够高级、不像真实摄影的提示词。

```text
使用 $image2-risk-debugger，这条提示词生成结果太保守，不够时尚张力，脸也偏幼，请修复成更成熟、高级、真实摄影感的版本：
……
```

## 参考词表怎么用

[参考词表.md](参考词表.md) 是一个风格和造型速查表，可以从里面挑选关键词，再交给 skill 整理成完整提示词。

推荐组合公式：

```text
面容 + 体态 + 衣着 + 气质 + 场景
```

例如：

```text
清冷面容 + 修长体态 + 白色长裙 + 高级疏离感 + 雪景
```

可以写成：

```text
使用 $image2-adult-glamour-studio，参考“清冷面容 + 修长体态 + 白色长裙 + 高级疏离感 + 雪景”，生成一条高级女性人像中文提示词。
```

也可以按用途选词：

- 风格：清冷仙气感、古典东方美、明艳大女主、法式优雅风、复古港风、街头酷飒风。
- 衣着：连衣裙、旗袍、汉服、西装套裙、针织裙、内衣、泳装、吊带睡裙、家居服。
- 气质：温柔、知性、成熟、清冷、明艳、优雅、健康、文艺、复古。
- 场景：窗边、咖啡馆、办公室、晚宴、海边、泳池、卧室、城市夜景、古风庭院。

## 输出内容

7 个 skill 默认输出简体中文，常见字段包括：

```text
意图摘要
已锁定参数
风险/质量说明
最终中文提示词
负面限制词
可选变化参数
```

`$image2-risk-debugger` 还会包含：

```text
修复策略
```

`$image2-character-poster-cover` 还会包含：

```text
随机封面/海报文字
```

`$image2-character-reference-sheet` 还会包含：

```text
设定图版面说明
```

其中 `最终中文提示词` 可以直接用于 Image2 / GPT Image 2。

## 边界

这些 skill 面向成年、原创、非露骨的时尚人像、魅力写真、Lookbook、商业摄影和角色设计提示词。

不会帮助生成或保留以下方向：

- 软色情或露骨色情目标
- 未成年感、幼态、学生元素与性感组合
- 真实人物、明星、网红、上传照片或私人形象的性化
- 偷拍、泄露、隐藏镜头、窥视、胁迫、醉酒、男友视角
- 身体局部主导构图、低机位身体凝视、透明裸露、私密暴露
- 绕过审查或规避安全系统的写法

如果原始请求包含这些内容，skill 会拒绝或改写相关部分，并把可支持的方向转成成熟、原创、非露骨的时尚/editorial 表达。
